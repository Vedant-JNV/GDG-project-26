## üî® Setup and instructions

> you need python and any IDE

Setup:

- Download the whole folder from the repo
- Open the main.py file from spacewars folder in your IDE
- and press F5 or simply run it
- can also run the .exe file downloaded from drive link

---

Gameplay instructions :-

- use arrow keys to move left and right
- use up arrow to fire bullets
- destroy as many spaceships as you can to score more
- press space bar to pause the game
- you lose when health bar becomes zero
